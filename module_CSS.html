CSS | Cascading Style Sheets

Le CSS (Cascading Style Sheets, ou Feuilles de style en cascade) est le langage utilisé pour mettre en forme un document HTML.

  • Il sépare le fond (le HTML) de la forme (le design).
  • Il permet de gérer les couleurs, les typographies, l'espacement et les mises en page.
  • Il permet d'adapter l'affichage pour les ordinateurs, tablettes et mobiles (Responsive Design).

CSS : Les Variables

📙 Qu'est-ce qu'une variable CSS ?

Aussi appelées Custom Properties (propriétés personnalisées), les variables CSS permettent de stocker une valeur spécifique (comme un code couleur, une taille de police, ou un espacement) pour la réutiliser facilement partout dans ton fichier de style.

Pourquoi les utiliser ?
Imaginons que la couleur principale de ton site (un bleu spécifique) soit utilisée à 50 endroits différents dans ton CSS. Si ton client te demande de changer ce bleu pour du vert, sans variable, tu devrais modifier les 50 lignes manuellement. Avec les variables, tu ne le changes qu'une seule fois au début de ton fichier !

📙 Comment déclarer une variable globale ?

On déclare généralement les variables globales dans la pseudo-classe :root, qui représente la racine absolue du document (la balise <html>).

  • Le nom d'une variable CSS doit toujours commencer par deux tirets --.
  • On respecte souvent la casse minuscule avec des tirets (kebab-case).
/* Déclaration des variables tout en haut du fichier CSS */
:root {
  --couleur-primaire: #1e3a5f;
  --couleur-accent: #e83e00;
  --fond-gris: #f0f2f5;
  --arrondi-bouton: 8px;
}

📙 Comment utiliser une variable ?

Pour appliquer la valeur que tu as stockée, tu dois utiliser la fonction CSS var() en mettant le nom de ta variable entre les parenthèses.

⚠️ Ne pas confondre avec le "var" de JavaScript !

En JavaScript, il est recommandé de ne plus utiliser var pour créer des variables (on préfère let et const). Cependant, en CSS, la fonction var() est obligatoire et totalement standard ! Elle signifie simplement "lis la valeur de cette propriété". Il n'y a aucun problème à l'utiliser. 

/* Utilisation des variables dans le reste du code */
h1 {
  color: var(--couleur-accent);
}

button {
  background-color: var(--couleur-primaire);
  border-radius: var(--arrondi-bouton);
  color: white;
  padding: 10px 20px;
}

body {
  background-color: var(--fond-gris);
}
Astuce de pro : Les valeurs de secours (Fallback)

La fonction var() permet de définir une deuxième valeur, séparée par une virgule. Cette valeur sera utilisée si la variable principale est introuvable ou mal écrite. C'est une excellente sécurité. 

.texte {
  /* Si --couleur-texte n'existe pas, le texte sera noir */
  color: var(--couleur-texte, black); 
}
