Introduction aux Langages de programmation Web

Un Langage | Un Rôle

HTML : langage de balises pour structurer le contenu (= squelette).

CSS : langage de règles de styles pour mettre en forme le contenu (l'apparence).

JavaScript : langage de programmation. Lui, il sert à rendre les pages web intéractives et il est principalement utilisé côté utilisateur.


Le HTML | La Structure

  • C'est le langage de base d'une page web.
  • Il utilise des balises (<h1>, <p>, <img>).
  • Sans HTML, rien n'apparaît sur une page web.
  • C'est le squelette du site.

Le CSS | Le Style

  • Sert à mettre en forme le HTML.
  • Gère les couleurs, polices, marges, alignements, animations.
  • Transforme une page web brute en quelque chose de beau et lisible.
  • Sépare le contenu (HTML) de l'apparence.

Le JavaScript| l'intéractivité

  • C'est un langage de programmation.
  • Permet de réagir aux actions de l'utilisateur :
    • ouvrir un menu,
    • valider un formulaire,
    • afficher un message
    • animerun élément :
  • Rend la page dynamique et vivante.
