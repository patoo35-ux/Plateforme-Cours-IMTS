JavaScript | Introduction

Le JavaScript (JS) est un langage de programmation exécuté par le moteur JS du navigateur. Il permet par exemple :

  • d'afficher un message lorsqu’on clique sur un élément
  • de faire apparaître ou disparaître des éléments
  • de vérifier un formulaire
  • de créer des animations
  • de charger du contenu sans recharger la page

JavaScript est le langage qui donne le comportement à une page web.

JavaScript | Le fonctionnement

Quand on ouvre une page web, le navigateur charge trois types de fichiers :

  • HTML — la structure
  • CSS — l’apparence
  • JavaScript — le comportement

JS est exécuté par le moteur JavaScript intégré au navigateur (comme V8 dans Chrome). Le moteur lit le code et l’exécute pour modifier la page, réagir aux clics, charger des données, etc.

JS peut être ajouté de deux façons :

  • directement dans le HTML avec la balise <script>
  • dans un fichier externe, par exemple script.js :
<script src="script.js" defer></script>

L’attribut defer indique au navigateur d’attendre que le HTML soit chargé avant d’exécuter le script.

| BONNE PRATIQUE | Vérifier que le script est bien lié au document HTML en plaçant un alert() temporaire dans script.js.

alert()

Lors du chargement de la page, une fenêtre pop‑up s’affiche, confirmant que le script est bien chargé. Ensuite, commenter l’instruction : 

// alert()

JavaScript | Les variables

📙 Qu’est‑ce qu’une variable ?

On peut imaginer une variable comme une boîte dans laquelle on range une information. Cette boîte possède :

  • un nom pour la retrouver
  • un contenu (la valeur stockée)

En JavaScript, une variable sert à stocker une information pour la réutiliser plus tard.

Les variables permettent de garder en mémoire une valeur utilisée lors de l’exécution du script.

📙 Comment créer une variable ?

On déclare une variable :

  • avec les mots‑clés let ou const
  • en lui donnant un nom : maVariable
  • en lui assignant une valeur avec =
  • en terminant par un point‑virgule ; (optionnel mais recommandé)
const maVariable = 1;
let age = 25;
  • let → valeur modifiable :
let age = 25;
age = 26; // OK
  • const → valeur verrouillée :
const pays = "France";
pays = "Italie"; // ❌ erreur

On utilise const pour les valeurs qui ne doivent pas changer, et let pour celles qui peuvent évoluer.

var existe encore dans les anciens codes, mais n’est plus recommandé.

Pour afficher une variable, on utilise la console du navigateur :

  • en tapant directement : maVariable
  • ou avec console.log() :
console.log(maVariable);

Pour modifier une variable : 

maVariable = 2;
console.log(maVariable); // 2

📙 Comment utiliser les variables ?

Les variables permettent de :

  • éviter les répétitions
  • rendre le code plus clair
  • stocker des données dynamiques
  • faire des programmes adaptatifs

Toujours donner un nom explicite à ses variables.

Exemple

JavaScript :

  2. met "Bienvenue !" dedans
  3. affiche son contenu avec console.log(message)

📙 Comment JavaScript gère la mémoire (en bref) ?

JavaScript gère la mémoire automatiquement grâce à un système appelé garbage collector, qui alloue, utilise puis libère la mémoire sans intervention du développeur.

Le cycle de la mémoire en JavaScript

JS suit toujours trois étapes :

  1. allouer la mémoire quand on crée une variable
  2. utiliser la mémoire quand le code s’exécute
  3. libérer la mémoire automatiquement quand elle n’est plus nécessaire, grâce au garbage collector

📙 Les Chaînes de Caractères — string

Elles permettent de stocker des mots, phrases ou caractères. Une chaîne peut contenir lettres, chiffres, symboles, mais ne peut pas commencer par un chiffre.

Les chaînes s’écrivent avec :

  • guillemets simples ' '
  • guillemets doubles " "
  • backticks ` ` (template strings)
const a = 'Salut les gens !';
const b = "Salut les gens !";
const c = `Salut les gens !`;

📙 Les Booléens — boolean

Un booléen représente une information qui n’a que deux états possibles :

  • true → vrai
  • false → faux

C’est le type utilisé pour exprimer une condition, un état logique ou une réponse binaire.

Exemples
  • Utilisateur connecté ? → true/false
  • Formulaire valide ? → true/false
  • Bouton activé ? → true/false
Comment déclarer un booléen ?
let estConnecte = true;
let aFiniExercice = false;
const vrai = true;
const jeSuisFaux = false;
À quoi servent les booléens ?

Ils sont essentiels pour prendre des décisions :

let estConnecte = true;

if (estConnecte) {
  console.log("Bienvenue !");
} else {
  console.log("Veuillez vous connecter.");
}

Les booléens sont souvent le résultat d’une comparaison. Les opérateurs de comparaison renvoient toujours true ou false.

Les valeurs truthy et falsy
  • truthy : considérées comme vraies
    • un nombre non nul
    • une chaîne non vide
    • un objet
    • un tableau

  • falsy : considérées comme fausses
    • 0
    • "" (chaîne vide)
    • null
    • undefined
    • NaN
    • false

Exemple :

if ("Nolwenn") {
  console.log("Ceci s'affiche");
}
// car "Nolwenn" est truthy

📙 Les Tableaux — array

Un tableau est une variable spéciale qui permet de stocker plusieurs valeurs dans un seul contenant, organisées dans un ordre précis.

Qu’est‑ce qu’un tableau ?

Un tableau est une liste ordonnée de valeurs. Chaque valeur possède une position appelée index.

Exemple 
let monTableau = ["bleu", "rouge", "vert", "jaune"];
console.log(monTableau);

// indexation :
0 → "bleu"
1 → "rouge"
2 → "vert"
3 → "jaune"

⚠️ En informatique, on compte à partir de 0.

Pourquoi utiliser un tableau ?

Parce qu’il permet de :

  • stocker plusieurs valeurs dans une seule variable
  • parcourir facilement une liste
  • ajouter, modifier ou supprimer des éléments
Exemple
  • Sans tableau :
let stagiaire1 = "Elisa";
let stagiaire2 = "Benjamin";
let stagiaire3 = "Patricia";
  • Avec tableau :
let stagiaires = ["Elisa", "Benjamin", "Patricia"];

Créer un tableau

  • Méthode avec crochets []
let couleurs = ["bleu", "rouge", "vert", "jaune"];
  • Tableau vide :
let couleurs = [];
  • Ajouter des éléments :
couleurs[0] = "bleu";
couleurs[1] = "rouge";
couleurs[2] = "vert";
couleurs[3] = "jaune";
  • Accéder aux éléments :
console.log(couleurs[0]); // bleu
console.log(couleurs[2]); // vert
  • Modifier un élément :
couleurs[3] = "orange";
console.log(couleurs); // ["bleu", "rouge", "vert", "orange"]
